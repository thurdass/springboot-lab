## ▶️ Running the project

Clone the repository:

```bash
git clone https://github.com/thurdass/springboot-lab.git
```

Go to the project directory:

```bash
cd springboot-lab
```

Run the application:

```bash
mvn spring-boot:run
```

The application will start on:

```
http://localhost:8080
```
